# Olist E-Commerce SQL & Tableau Analysis
Analysis of 100K+ real Brazilian e-commerce orders using SQL and Tableau. 

## Business Questions Answered
1. Which product categories generate the most revenue?
2. Which states have the hightest customer concentration?
3. Which states have the slowest delivery times?
4. What percentage of orders are dleivered late?
5. Which product categories have the best/worst reviews?

## Tools
- SQL (PostgreSQL): GitHub Codespaces
- Tableau Public: dashboard link below

## Key Findings:
- health_beauty is the #1 revenue category at $1,258,681.
- 8.1% of all delivered orders arrive late.
- Roramia (RR) has the higest average delivery time at 29.4 days.
